# Pruebas Manuales SGCM

## CP01 — Registro exitoso
Descripción: Validar que un usuario pueda registrarse correctamente.
Precondiciones: Backend y frontend funcionando.
Pasos:
1. Abrir frontend.
2. Ir a Registro.
3. Llenar formulario.
4. Enviar.
Resultado esperado: Aparece pantalla de “Mis citas”.

## CP02 — Login incorrecto
Descripción: Validar error cuando las credenciales no son válidas.

## CP03 — Crear cita
Descripción: Validar que un paciente pueda agendar una cita correctamente.
